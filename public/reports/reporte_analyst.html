Análisis Sugeridos

  3. Hipótesis: Las campañas con mayor CPL tienen menor retorno de inversión.
  4. Query (ejemplo):

sql


SELECT ic.campana,


ic.gasto,


SUM(ld.leads) AS leads_totales,


ic.gasto / NULLIF(SUM(ld.leads),0) AS CPL_campana


FROM inversiones_campanas ic


JOIN leads_diarios ld ON ld.fecha BETWEEN ic.fecha_inicio AND ic.fecha_fin


GROUP BY ic.campana, ic.gasto


ORDER BY CPL_campana DESC;


  1. Impacto del número de intentos en la tasa de conversión
  2. Tabla: `llamadas_detalle` (lead_id, intentos, fecha_ultimo_intento) + tabla `conversiones` (lead_id, fecha_conversion).
  3. Hipótesis: Leads con > 7 intentos tienen una probabilidad de conversión significativamente menor.
  4. Query:

sql


SELECT CASE WHEN l.intentos > 7 THEN '>7' ELSE '<=7' END AS rango_intentos,


COUNT(DISTINCT l.lead_id) AS total_leads,


COUNT(DISTINCT c.lead_id) AS leads_convertidos,


COUNT(DISTINCT c.lead_id)::float / COUNT(DISTINCT l.lead_id) AS tasa_conversion


FROM llamadas_detalle l


LEFT JOIN conversiones c ON c.lead_id = l.lead_id


GROUP BY rango_intentos;

  4. Query:

sql


SELECT


CASE


WHEN l.tiempo_entre_intentos_min <= 30 THEN '≤30min'
